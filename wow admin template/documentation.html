Documentation

Inroduction

This is a premium admin dashboard template with Material design. It is fully responsive admin dashboard template built with Bootstrap 4 Framework, HTML5 and CSS3, Media query. It has a huge collection of reusable UI components and integrated with latest jQuery plugins. It can be used for all type of Web applications like custom admin panel, app backend, CMS or CRM.

How to change colors?

We have made Color Variables in main.css File which you can find in assets/css/main.css

  • --primary-color
  • --secondary-color
  • --dark-color
  • --light-color

just by changing the primary variable color you can change whole theme color.

Use below classes for Background Colors
  • bg-primary
  • bg-secondary
  • bg-dark
  • bg-light
  • bg-success
  • bg-danger
  • bg-warning
Use below classes for Text Colors
  • text-primary
  • text-secondary
  • text-dark
  • text-light
  • text-success
  • text-danger
  • text-warning
How to edit all the files ?

All index files represent that how can you use all the widgets,tables,charts etc. Other html files are created using there original names like forms_uploads.html, charts_bar.html etc. You can easily edit them.

How to change Navigation Color?

Your can use three Color Combinations for Navigation Sidebar: 1. With white background 2. With primary background 3. With dark background

Add .bg-primary class to 'nav' tag to get navigation with primary background.

Add .bg-dark class to 'nav' tag to get navigation with primary background.

Add .bg-white class to 'nav' tag to get navigation with primary background.
